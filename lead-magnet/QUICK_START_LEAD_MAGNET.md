## ðŸ†˜ Troubleshooting

**"I'm not getting downloads"**

- Test form submission
- Simplify form (just name + email)

- Improve follow-up email copy

- Use Chrome for printing
- Check "Background graphics" is ON
- Set margins to None
- Verify scale is 100%
